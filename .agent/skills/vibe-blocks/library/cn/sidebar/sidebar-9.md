# Sidebar 9

## Metadata
- **Category**: Sidebar
- **Objective**: Sidebar with an 'inset' layout, featuring breadcrumb navigation and a collapse trigger.
- **Use Case**: Applications where the sidebar is visually separated from the main content area (inset), providing clear hierarchy via breadcrumbs.
- **Visual Style**: Inset variant with a clean header containing navigation aids.
- **Interactivity**: Expandable/collapsible sidebar with breadcrumb integration.

## Description
This sidebar variation utilizes the 'inset' variant to create a distinct visual separation between the navigation menu and the content area, enhanced by breadcrumbs for better spatial orientation.

const NavMenuItem = ({ item }: { item: NavItem }) => {
  const Icon = item.icon;

  if (!hasChildren) {

    );

  return (
    <Collapsible
      asChild
